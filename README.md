# elasticsearch-snippets

This extension provides code JSON code snippets for the more commonly used queries in the Elasticsearch Query DSL.

## Features

![in action](images/action.gif)

* Filtered query
* Nested query
* Bool query
* Term query
* Range query
* Exists query
* Ids query
* Match query (boolean)
* Common Terms query
* Function Score query
